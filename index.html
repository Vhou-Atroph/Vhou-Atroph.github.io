TT-Damage-Calculator

The Tootown Damage Calculator is a desktop program written in Python and Rust that allows for calculating damage dealt to a single cog, assuming all gags hit.

Alongside normal cogs, the calculator is able to evaluate damage against cogs with a defense buff in Field Offices, as well as level 9-12 V2.0 cogs, which have reinforced plating.

The program is available for download as a Windows executable on GitHub at https://github.com/Vhou-Atroph/TT-Damage-Calculator/releases/latest. In the future, it will also be available on Pypi.
